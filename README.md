## Descripción
Este es el Trabajo Práctico 1 para la materia Frontend del IFTS N.29. Es un proyecto web desarrollado utilizando HTML5 semántico, CSS3 con diseño adaptable (Responsive Web Design) y funcionalidades interactivas con JavaScript. 

El proyecto fue desarrollado en equipo con un entorno de trabajo colaborativo utilizando Git y GitHub.

## Integrantes del Equipo
* Maria Lopez - https://github.com/Marialopez2020
* Matias Contreras - https://github.com/Matihp

## Tecnologías Utilizadas
* **HTML5**: Uso de etiquetas semánticas para mejorar la accesibilidad.
* **CSS3**: Maquetación con Flexbox, modelo de cajas y diseño adaptable con Media Queries.
* **JavaScript**: Manipulación del DOM para interactividad en portada y tarjetas.

## Estructura de Archivos
* `/`: index.html, maria.html, matias.html, bitacora.html y README.md.
* `/css`: estilos.css.
* `/js`: principal.js y tarjetas.js.
* `/img`: imágenes y avatares de los integrantes.

## Guía de Estilos
* **Colores**: Fondo `#f4f4f9`, Texto `#333333`, Primario `#007bff`, Secundario `#6c757d`.
* **Tipografía**: Arial, sans-serif.
* **Iconografía**: Se utilizaron avatares generados por IA.

## Funciones JavaScript
* **Portada**: Función en `principal.js` que agrega un EventListener al evento `submit` del formulario de contacto para validar el envío.
* **Tarjetas**: Función en `tarjetas.js` que permite cambiar el color de fondo de la página (botón de Maria) y el color de fondo de la tarjeta (botón de Matias) con colores aleatorios.
