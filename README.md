# TP1 Proyecto Web

## Descripción
Este es el Trabajo Práctico 1 para la materia Frontend del IFTS N.29. Es un proyecto web desarrollado utilizando HTML5 semántico, CSS3 con diseño adaptable (Responsive Web Design) y funcionalidades interactivas con JavaScript. 

El proyecto fue desarrollado en equipo con un entorno de trabajo colaborativo utilizando Git y GitHub.
